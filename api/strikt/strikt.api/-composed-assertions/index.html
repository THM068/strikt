class ComposedAssertions<T>

Allows assertions to be composed, or nested. This class is the receiver of the lambda passed to AssertionContext.compose.

Functions

assert

 fun assert(description: String, assertion: AssertionContext<T>.() -> Unit): Assertion<T>

Evaluates a composed assertion on the original subject. This creates a new assertion in the composed context using the same subject as the overall assertion. This is useful because it allows for the overall assertion to contain much more detail in any failure message.
